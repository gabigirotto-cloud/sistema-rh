Sincronização com Google Drive e backup

🔗 Google Drive — Sincronização em nuvem
Os dados são salvos em rh-dados.json no seu Google Drive. Qualquer computador autenticado com a mesma conta acessa e atualiza os mesmos dados. O app funciona offline normalmente — sincroniza quando a internet voltar.
Como configurar (uma vez só):
1. Acesse console.cloud.google.com e crie um projeto
2. Ative a Google Drive API em APIs e serviços → Biblioteca
3. Crie credencial: OAuth 2.0 → Aplicativo Web
4. Em "Origens autorizadas" adicione a URL deste app (ex: https://seuusuario.github.io/sistema-rh)
5. Copie o Client ID e cole abaixo
